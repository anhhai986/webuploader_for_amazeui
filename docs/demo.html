基于百度webuploader的AMUI上传组件（支持移动端）
点击上传队列中的图片，可将其移出队列中。

相应代码:

<div data-am-webuploader-simple="{id:'demo', name:'thumb', pick:{id:'#demo'}}"></div>

相应代码:

<div data-am-webuploader-simple="{id:'more', name:'more[]',pick:{id:'#more', multiple:true}}"></div>
常用的文件格式可以参考：点击访问
图片格式可以参考：accept:{title:'Images', extensions:''gif,jpg,jpeg,bmp,png', mimeTypes:'image/*'}

相应代码:

<div data-am-webuploader-simple="{id:'file', name:'file',pick:{id:'#file'}, accept:{title:'File', extensions: 'txt,doc,docx,xls,xlsx', mimeTypes:'text/plain,application/msword,application/vnd.ms-excel,application/vnd.openxmlformats-officedocument.wordprocessingml.document,application/vnd.openxmlformats-officedocument.spreadsheetml.sheet' }}"></div>
一般默认的服务器上传大小为2M，使用分片上传可以有效避免文件上传限制。但后端需要做相应的配置，具体请自行参考本文中的fileupload.php
分片上传相关参数: chunked:true, chunkSize: 50 * 1024, chunkRetry: 10

相应代码:

<div data-am-webuploader-simple="{id:'chunked', name:'chunked[]', chunked:true, chunkSize: 50 * 1024, chunkRetry: 10, pick:{id:'#chunked', multiple:true}}"></div>
若觉得HTML声明对象方式太过冗长，可以尝试使用JS方式声明。

相应代码:

<div id="js-init"></div>
<script>
    $(function(){
        AMUIwebuploader.init('js-init', {
            id:'jscreate',
            name:'jscreate',
            pick:{
                id:'#jscreate'
            }
        });
    })
</script>
